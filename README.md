# ADS - Resolution of Algorithms and Data Structures Problems

<br>

## Description
Resolution of problems using different approaches, involving complexity analysis, data structures (Merkle, AVL, Splay Tree) and sorting algorithms (Shell, Quick, Insertion, Radix Sort).

### Technologies:
- Java
- C

<br>

## Information
This project contains 4 sub-sections:
- Proj1 - Problem Solving
- Proj2 - Complexity Analysis
- Proj3 - Data Structures (Merkle, AVL, Splay Tree)
- Proj4 - Sorting Algorithms (Shell, Quick, Insertion, Radix Sort)
